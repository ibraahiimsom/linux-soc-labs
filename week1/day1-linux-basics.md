# Day 1 – Linux Basics

## Objective
Learn how to navigate a Linux system and understand key directories.

---

## Commands Used

- pwd → shows current directory
- ls → lists files
- ls -la → shows all files including hidden ones
- cd → change directory
- cd .. → go back one level
- cd / → go to root directory
- cd ~ → go to home directory

---

## Lab Steps

1. Checked current directory using:
   pwd

2. Listed files:
   ls

3. Viewed hidden files:
   ls -la

4. Navigated to root:
   cd /

5. Explored system folders:
   cd /var/log

6. Returned to home:
   cd ~

## Key Learning

Knowing your location in the system is important because it helps you avoid mistakes and understand where files are stored.

---

## SOC Insight

The /var/log directory is important because it stores system logs that can show login attempts, errors, and suspicious activity.
